## 1. Start the AI "Brain" (Ollama)
First, make sure Ollama is actually running in the background.

*   Look for the **Ollama icon** (the llama head) in your Windows System Tray (bottom right near the clock).
*   If you don't see it, search for "Ollama" in your Start Menu and open it.
*   **Verification:** Open a terminal and type `ollama list`. You should see `llama3.2` and `nomic-embed-text` in the list.

---

## 2. Launch the Backend (FastAPI)
Open your first terminal window in your root `LOREBAIT` folder.

1.  **Activate your environment:**
    ```powershell
    .\venv\Scripts\activate
    ```
2.  **Run the server:**
    ```powershell
    uvicorn main:app --reload --port 8000
    ```
    *You should see a message saying "Uvicorn running on http://127.0.0.1:8000". Leave this window open.*
